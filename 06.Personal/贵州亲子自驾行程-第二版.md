# 🚗 贵州亲子自驾行程 · 第二版（柳州中转方案）

**车辆：** 蔚来 ES8（102kWh）
**日期：** 2026.7.22（三）— 7.31（五），共 10 天
**人员：** 两大两小（亲子）
**起点：** 佛山南海桂城街道

| 场景 | 续航 |
|------|------|
| 高速巡航 | ~420km |
| 贵州山区高速 | ~360km |
| 满电时长 | 45min（10%→100%） |
| 途中快充策略 | 跑约 350km → 充 30min 至约 80% → 继续 |

## 📅 每日行程

### D1｜7.22（三）佛山桂城 → 柳州（400km）

| 时间 | 事项 |
|------|------|
| 09:00 | 满电出发 |
| 13:30 | 到**柏曼酒店（柳州白沙大桥游船码头店）**入住，**慢充过夜** |
| 下午 | **柳州工业博物馆**（亲子推荐，有真火车可爬）→ **窑埠古镇** |
| 傍晚 | 正宗**螺蛳粉**（发源地） |

---

### D2｜7.23（四）柳州 → 贵阳观山湖（400km）

| 时间 | 事项 |
|------|------|
| 09:00 | 满电出发 |
| 13:30 | 到**全季酒店（观山湖金融城店）**入住，**连住 3 晚** |
| 下午 | **贵州省博物馆**（近酒店）或 **黔灵山公园**（停车延安西路坐地铁） |
| 傍晚 | 民生路小吃 → 甲秀楼 → 青云市集 → 白宫双子塔夜景 |

> **选店理由：** 近贵州省博物馆、贵阳欢乐世界，停车至延安西路地铁口换乘去黔灵山公园，去民生路、喷水池、太平路夜市、甲秀楼、青云市集等市中心景点也方便。

---

### D4｜7.25（六）贵阳 → 织金洞 → **黄果树住**

| 时间 | 事项 |
|------|------|
| 07:40 | **退房**，满电出发 |
| 09:10–11:30 | **织金洞**游览 2.5h（洞内 14℃，带薄外套，不可推婴儿车，门口可寄存） |
| 11:30–12:30 | 景区门口午餐 |
| 12:30–14:30 | 到黄果树，入住**锦江丽怡酒店（安顺镇宁黄果树大道店）**，**慢充过夜** |
| 下午/傍晚 | 酒店休息 / 周边逛逛 |
| ⚠️ | **当天不进景区**，养足精力明日再玩 |

---

### D5｜7.26（日）黄果树 **全天**

| 时间 | 事项 |
|------|------|
| 08:30 | 进景区 |
| 上午 | **陡坡塘**（西游记取景，1h）→ **大瀑布+水帘洞**（2h，雨衣必备） |
| 午间 | 景区午餐 或 回酒店午休 |
| 下午 | **天星桥下半段**（务必走后半段，银链坠潭瀑布是精华，2-3h） |
| 傍晚 | 酒店附近晚餐 |
| 住宿 | 锦江丽怡酒店（续住） |

---

## 🎒 亲子必备清单

- 溯溪鞋（小七孔水上森林必穿）
- 速干换洗衣物 × 多套（玩水一天换两三次）
- 折叠雨衣 × 3（黄果树水帘洞 + 贵州 7 月多雨）
- 高倍防晒 + 驱蚊液
- 织金洞薄外套（洞内 14℃）
- 现金若干（天眼山顶无信号）

## ⚠️ 注意事项

- 门票：黄果树、荔波小七孔、天眼提前在「一码游贵州」小程序预约
- 织金洞：无法推婴儿车，门口可付费寄存
- 天眼：任何电子设备不得带入，寄存后上山
- 订房确认：预订时跟店家确认有慢充桩或 220V 插座可用

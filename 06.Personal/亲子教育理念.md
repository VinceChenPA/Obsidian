# 亲子教育理念

## 核心观点

1. **鼓励式教育** — 不对孩子指责、不翻旧账，建立信任关系
2. **观察式与倾听式** — 以观察和倾听为主，让孩子愿意主动沟通
3. **接纳而非打压** — 对青春期恋爱等敏感话题保持开放态度，承认正常的成长需求
4. **性教育比严防死守更有效** — 让孩子有自我保护意识，对自己负责
5. **孩子的人生终究要自己负责** — 打压没有用，引导比控制更重要
